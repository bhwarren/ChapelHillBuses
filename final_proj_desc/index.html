Final Project, Comp 426: Nextbus Redesign

Team: Bo Warren, Alex Parker, Aaron Brown

Description

      We plan to re-design the Next Bus website so that we can list routes based solely on stops, rather than just looking at times by selecting a route and then a stop. This should ease the search for finding multiple buses that go to the same place, and allow the arrival times to be compared.

Details

      We will start out with a mockup, and expand the user interface from there. The user should be able to select just a stop or just a route and have results displayed which matches the input. The backend will be a realational database with list of routes, stops, and times, which can be queried and displayed back to the user once a specific stop is selected.
